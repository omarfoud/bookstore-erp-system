# ERP System Project

## ملخص المشروع (Project Summary)

هذا مشروع نظام تخطيط موارد المؤسسات (ERP) متكامل تم تطويره باستخدام بنية العميل-الخادم (Client-Server Architecture) لإدارة مختلف جوانب العمليات التجارية.

### الهيكل التقني (Technical Architecture)

**الواجهة الأمامية (Frontend):**
- **التقنية:** React 19.2.1 مع JavaScript
- **مكتبات واجهة المستخدم:** Chakra UI 2.8.2
- **إدارة الحالة:** React Hooks
- **التوجيه:** React Router DOM 7.10.1
- **الرسوم البيانية:** Recharts 3.5.1
- **الأيقونات:** React Icons 5.5.0

**الواجهة الخلفية (Backend):**
- **التقنية:** Flask Framework مع Python
- **قاعدة البيانات:** MySQL مع mysql-connector-python
- **المصادقة:** JWT Tokens مع Flask-Bcrypt
- **التواصل بين الخدمات:** CORS للتواصل بين الواجهة الأمامية والخلفية

### الوحدات الرئيسية (Main Modules)

1. **وحدة المصادقة (Authentication Module)**
   - تسجيل الدخول والخروج
   - إدارة المستخدمين
   - حماية المسارات باستخدام JWT

2. **وحدة المخزون (Inventory Module)**
   - إدارة المنتجات
   - تتبع الكميات
   - إدارة الفئات

3. **وحدة المبيعات (Sales Module)**
   - إدارة الطلبات
   - تتبع المبيعات
   - تقارير المبيعات

4. **وحدة المالية (Finance Module)**
   - إدارة الحسابات
   - تقارير مالية
   - تتبع الإيرادات والمصروفات

5. **وحدة الموارد البشرية (HR Module)**
   - إدارة الموظفين
   - بيانات الموظفين
   - إدارة الرواتب

6. **وحدة الإدارة (Admin Module)**
   - إدارة النظام
   - إعدادات النظام
   - تقارير شاملة

### المميزات الرئيسية (Key Features)

- **واجهة مستخدم حديثة:** تصميم متجاوب وسهل الاستخدام
- **أمان متقدم:** تشفير كلمات المرور ومصادقة JWT
- **تقارير متقدمة:** رسوم بيانية تفاعلية باستخدام Recharts
- **بنية معيارية:** كود منظم وسهل الصيانة
- **دعم متعدد اللغات:** واجهة باللغة العربية

### كيفية تشغيل المشروع (How to Run)

**الواجهة الخلفية:**
```bash
cd backend
pip install -r requirements.txt
python app.py
```

**الواجهة الأمامية:**
```bash
cd frontend
npm install
npm start
```

### المتطلبات (Prerequisites)

- Python 3.7+
- Node.js 14+
- MySQL Server
- Git

### إعداد قاعدة البيانات (Database Setup)

يحتوي المشروع على ملف `bookstore_erp.sql` في مجلد `backend/` الذي يحتوي على:
- هيكل قاعدة البيانات الكامل
- الجداول اللازمة لجميع الوحدات (الموظفين، المخزون، المبيعات، الحضور، إلخ)
- بيانات تجريبية للاختبار

**لاستيراد قاعدة البيانات:**
```sql
-- استيراد الملف في MySQL
mysql -u username -p bookstore_erp < backend/bookstore_erp.sql
```

**الجداول الرئيسية:**
- `users` - بيانات المستخدمين
- `employees` - معلومات الموظفين
- `attendance` - سجل الحضور والغياب
- `products` - منتجات المخزون
- `orders` - طلبات المبيعات
- `finance_records` - السجلات المالية

## مقدمة للمعيدة (Introduction for Instructor)

### نبذة عن المشروع

هذا المشروع هو تطبيق لنظام تخطيط موارد المؤسسات (ERP) مصمم لإدارة وتوحيد العمليات التجارية المختلفة في منظمة واحدة. تم تطوير المشروع كجزء من متطلبات المقرر الدراسي، حيث يهدف إلى تطبيق المفاهيم النظرية في بيئة عملية حقيقية.

### الأهداف التعليمية

1. **تطبيق بنية العميل-الخادم:** فهم كيفية بناء تطبيقات ويب حديثة باستخدام تقنيات متخصصة للواجهة الأمامية والخلفية
2. **إدارة قواعد البيانات:** التعامل مع قواعد البيانات العلائقية وتصميم المخططات المناسبة
3. **تطوير واجهات المستخدم:** إنشاء واجهات مستخدم تفاعلية وسهلة الاستخدام
4. **الأمان والمصادقة:** تطبيق أفضل الممارسات في أمان التطبيقات
5. **هندسة البرمجيات:** تطبيق مبادئ التصميم الجيد والكود القابل للصيانة

### التقنيات المستخدمة

تم اختيار التقنيات بعناية لتعكس الممارسات الصناعية الحديثة:
- **React:** مكتبة شائعة لبناء واجهات المستخدم التفاعلية
- **Flask:** إطار عمل خفيف ومرن للواجهة الخلفية
- **MySQL:** نظام قواعد بيانات علائقي موثوق
- **JWT:** معيار صناعي للمصادقة في التطبيقات الحديثة

### التحديات والحلول

أثناء التطوير، تم مواجهة عدة تحديات مثل:
- تصميم مخطط قاعدة بيانات مرن
- إدارة الحالة في تطبيقات React
- تأمين واجهات API
- تحسين أداء التطبيق

تم حل هذه التحديات باستخدام أفضل الممارسات والتقنيات المناسبة لكل مشكلة.

### النتائج

النتيجة النهائية هي نظام ERP متكامل يوفر:
- واجهة مستخدم حديثة وسهلة الاستخدام
- بنية تحتية قابلة للتوسع
- أمان متقدم لحماية البيانات
- تقارير وتحليلات شاملة

هذا المشروع يمثل تطبيقاً عملياً للمفاهيم النظرية التي تم دراستها خلال الفصل الدراسي، ويظهر القدرة على بناء تطبيقات ويب احترافية باستخدام التقنيات الحديثة.
